# 📦 Smart Warehouse Management System

A C++ based inventory management solution developed as part of Computer Engineering coursework at the University of Damascus. This system is designed to streamline warehouse operations, including stock tracking, item management, and data organization.

## 🚀 Features
* Product Management: Add, delete, and update warehouse items efficiently.
* Search Functionality: Quickly locate items using Unique ID or Name.
* Inventory Reports: Generate a full list of available stock with quantities and status.
* Object-Oriented Design (OOP): Built using classes and encapsulation for modular and maintainable code.

## 🛠️ Tech Stack
* Language: C++
* IDE: Visual Studio 2022
* Version Control: Git & GitHub

## 📂 Project Structure
* Smart Warehouse Mangament System.cpp: The main source code containing the logic.
* README.md: Project documentation and guide.

## 💻 How to Run
1. Clone the repository or download the source file.
2. Open the file in Visual Studio or any C++ compiler.
3. Build and Run the project.
4. Follow the on-screen console menu to interact with the system.
